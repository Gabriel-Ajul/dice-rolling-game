# dice-rolling-game
This is a console-based C++ dice rolling game simulation where users can roll a six sided die repeatedly.

## Features

- 1-6 die roll value range
- Simple roll choice selection
- Displays roll value
- Random number generator used to randomly generate numbers for dice roll
- Re-roll feature using loops

## How to Run Simulation
1. Clone the repository: https://github.com/Gabriel-Ajul/dice-rolling-game.git

2. Compile: g++ dice_rolling_game.cpp -o dice_rolling_game

3. Run ./dice_rolling_game

## Techniques Employed:

- OOP
- Loops
- Basic I/O
- RNG
- GitHub version control
